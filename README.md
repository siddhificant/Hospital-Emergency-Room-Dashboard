# Hospital Emergency Room Dashboard

## Project Overview
This project is an interactive Hospital Emergency Room Dashboard built in Power BI to analyze emergency room operations and patient trends.

The dashboard provides insights into patient admissions, wait times, referrals, satisfaction scores, demographics, and peak hospital hours to support operational decision-making.

---

## Objectives
- Monitor emergency room performance metrics
- Analyze patient flow and wait times
- Track admission and referral trends
- Identify peak hours and patient patterns
- Provide detailed patient-level insights

---

## Dashboards Included

### 1. Monthly View
Tracks monthly ER performance including:
- Number of patients
- Average wait time
- Patient satisfaction score
- Patients referred
- Admission status
- Age group distribution
- Gender analysis
- Race analysis
- Department referrals
- Patient volume by day and hour

### 2. Consolidated View
Provides aggregated insights across a customizable date range:
- KPI trends
- Referral analysis
- Wait time analysis
- Demographic analysis
- Operational performance overview

### 3. Patient Details View
Detailed patient-level table containing:
- Patient ID
- Gender
- Age
- Admission date
- Race
- Wait time
- Department referral
- Admission status
